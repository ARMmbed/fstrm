fstrm_file_writer

Detailed Description

fstrm_file_writer is a concrete implementation of the fstrm_writer interface. It writes Frame Streams content into the regular file specified by fstrm_file_writer_options_set_file_path. Note that the opened file will be truncated.

The fstrm_file_writer implementation is generally used with fstrm_io by creating an fstrm_file_writer_options object, setting the file_path parameter on this options object, and passing it to fstrm_io_options_set_writer(), as in the following example:

const char *file_path = "/path/to/output.fs";
struct fstrm_file_writer_options *fwopt;
fwopt = fstrm_file_writer_options_init();
fstrm_file_writer_options_set_file_path(fwopt, file_path);
struct fstrm_io_options *fopt;
fopt = fstrm_io_options_init();
fstrm_io_options_set_writer(fopt, fstrm_file_writer, fwopt);
char *errstr = NULL;
struct fstrm_io *fio;
fio = fstrm_io_init(fopt, &errstr);
fstrm_io_options_destroy(&fopt);
fstrm_file_writer_options_destroy(&fwopt);
if (fio != NULL) {
// fio is ready to accept data now.
} else {
fprintf(stderr, "Error: fstrm_io_init() failed: %s\n", errstr);
free(errstr);
}
